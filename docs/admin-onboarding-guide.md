# Admin Onboarding Guide

Use this after the nonprofit’s deployment is live.

## First login

1. Open the deployment URL.
2. Sign in with the approved admin email.
3. Complete the setup guide at `/setup`.

## Setup checklist

### Branding
- set organization name
- set subtitle and welcome text
- choose colors
- upload logo and favicon

### Organization details
- add support email or phone
- set the support CTA text
- review login copy

### Access
- approve the first staff emails
- confirm how client portal emails will be managed
- review allowlist guidance

### Starter data
- decide whether to import via CSV or enter records manually
- download the client CSV template if importing

### Launch
- review the admin dashboard
- verify staff login
- verify client portal login

## Ongoing admin tasks

- manage approved emails in `/admin`
- review audit logs
- export client or service data when needed
- update branding or support contact from `/setup`
