# Staff Quick-Start

## What staff can do

- create clients
- search the client directory
- open client profiles
- log services
- manage appointments
- use semantic note search

## Daily workflow

1. Sign in.
2. Review `/dashboard/staff` for today’s appointments and recent client activity.
3. Add new clients from `/clients/new`.
4. Open client profiles from `/clients`.
5. Log services directly on the client profile.
6. Review and add appointments in `/schedule`.

## Important guardrails

- staff cannot access `/admin`
- staff cannot export data
- staff cannot approve new accounts
- client portal users do not see internal note bodies
