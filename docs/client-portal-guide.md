# Client Portal Guide

The client portal is intentionally simple and read-only.

## What clients can see

- current case status
- upcoming appointments
- recent support activity metadata
- last interaction date
- support contact information

## What clients cannot see

- internal case notes
- admin or staff dashboards
- client directory
- exports
- audit logs

## Support reminder

If appointment details look wrong or a situation changes, the client should contact the organization using the support information shown in the portal.
